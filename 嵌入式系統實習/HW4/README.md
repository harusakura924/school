## Buzzer 練習  
**利用#define指令，定義Buzzer的開啟與關閉**  
- BuzzerOn： Buzzer的開啟  
- BuzzerOff： Buzzer的關閉  
  
**Buzzer應用於紅綠燈程式中**(HW #3)  
- 當黃燈與左轉燈(或行人燈)閃爍時，Buzzer同時開啟與關閉  
-> 即，燈亮出聲；燈滅閉聲。  
-> DrvSYS_Delay(300000);
